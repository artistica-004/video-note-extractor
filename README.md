# 🎬 Video Note Extractor

Convert any YouTube video into organized notes instantly using AI!

## Features
- 📝 Organized Notes & Summary
- ⏱️ Important Timestamps
- ✅ Action Items & Tasks
- 📄 Full Transcript

## How to Use
1. Paste any YouTube URL
2. Click Extract Notes
3. View your notes in tabs
4. Download as .txt file
